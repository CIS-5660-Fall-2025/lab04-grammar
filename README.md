## 3. Custom plant

For my custom plant I tried mimicing a fir tree.

In my rules below, A is used to "seed" itself, by placing an I (branch) and then another seed A rotated by 45 degrees each time. This rotation is around the z axis.

For the branches I, if it is the first iteration, it sproutes forward, then places a branch incrementing its counter. Once the counter reaches a value of 2, it keeps growing, but then places down 4 seeds at 90 degree angles with one another at the same spot along the branch.
